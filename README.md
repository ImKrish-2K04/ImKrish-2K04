## 🚀 Featured Projects

### 🧠 AI PDF Summarizer — SaaS
> Full SaaS product with subscription billing and AI-powered PDF summarization

- Extracts and summarizes PDF content using **OpenAI**, **Gemini AI**, and **LangChain's WebPDFLoader**
- File uploads handled via **UploadThing** (cloud storage)
- Auth via **Clerk**, database via **Neon DB + Drizzle ORM**
- **Stripe** integration with 3 subscription tiers: Monthly Basic, Monthly Pro, Yearly Pro
- Built with **Next.js**, **shadcn/ui**, **TypeScript**

---

### 🎤 Mock AI Interview Platform
> AI-powered interview prep platform with real-time feedback

- Users can sign up, pick a role, and practice mock interviews with an AI interviewer
- Powered by **Gemini API** for dynamic question generation and feedback
- Built with **React.js**, **Firebase + Firestore**, **Clerk Auth**, **shadcn/ui**

---

### 🔐 ts-auth-api — Production-Grade Auth System
> A complete authentication & authorization API built from scratch in TypeScript

- **JWT** access + refresh tokens with rotation and reuse detection
- **Google OAuth** via Passport.js
- **2FA / TOTP** with `otplib` + QR code generation
- Role-based access control (RBAC), Zod validation, rate limiting
- Stack: **Node.js + Express + MongoDB + TypeScript**

---

### 🖥️ Classroom Dashboard — PERN Stack
> A full-stack classroom management dashboard

- Built with **PostgreSQL + Express + React + Node.js + Arcjet + Refine Core + Site24x7 + CodeRabbitAI**
- Clean UI for managing classroom data with full CRUD operations
